# Juego de la Serpiente en Python

Proyecto sencillo del clásico juego Snake usando Python y Pygame.

## Requisitos

Python 3
Pygame

## Instalación

pip install pygame

## Ejecutar

python snake.py

## Controles

Flechas del teclado para mover la serpiente.
